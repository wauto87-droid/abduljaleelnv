CMS Setup Guide

How to activate your "Thoughts" Page

Connect your website to Google Sheets to post updates instantly.

Step 1: Create the Sheet

1. Go to Google Sheets and create a new sheet.

2. Rename the tab (at the bottom) to content.

3. In the first row, add these exact headers:

id, type, title, description, image_url, created_date, status

Step 2: Add the Backend Code

1. In the Sheet, go to Extensions > Apps Script.

2. Delete any existing code.

3. Copy the code from the file assets/code.gs in your project folder and paste it there.

4. Save the project (Ctrl+S).

Step 3: Deploy as Web App

1. Click the blue Deploy button (top right) > New deployment.

2. Click the gear icon > Web app.

3. Set Who has access to Anyone. (Important!)

4. Click Deploy.

5. Copy the Web app URL (it ends with /exec).

Step 4: Connect the Website

1. Open the file js/cms.js in your project folder.

2. Find the line: const API_URL = 'MOCK_DATA';

3. Replace 'MOCK_DATA' with your Web App URL inside quotes.

Example:

const API_URL = 'https://script.google.com/macros/s/AKA.../exec';

Step 5: Add Content

Go back to your Google Sheet and add rows!

- type: quote, post, or image
- status: published (only these will show)
- image_url: Link to any image (optional)
